Landscape Quick Start Guide

Guía de inicio para crear, esculpir y pintar terrenos con el sistema Landscape de UE5.

1

Herramientas y Modos de Landscape

El sistema Landscape permite crear terrenos exteriores extensos optimizados para mantener un framerate estable en múltiples plataformas. Todas las herramientas están accesibles desde el menú Modes → Landscape.

Panel principal de la herramienta Landscape con los tres modos disponibles

PaintPintar capas de textura sobre el terrenoAplicar grass, roca, tierra o nieve por zonas

Crear un Nuevo Landscape

Antes de crear el terreno necesitas un proyecto con una plantilla base. La plantilla First Person es conveniente para explorar el Landscape a nivel del suelo durante el desarrollo.

ℹ
Starter Content

Asegúrate de incluir el Starter Content al crear el proyecto; lo usarás para los materiales de prueba iniciales.

Paso 2 — Crear un nuevo nivel

Una vez dentro del proyecto, crea un nivel limpio con File → New Level → Empty Level para partir desde cero sin actores extra.

Menú File → New Level en el editor de UE5

Paso 3 — Preparar el nivel

  • 1Elimina el Floor predeterminado si lo hay: selecciónalo y presiona Delete.
  • 2Desplaza el Player Start ligeramente hacia arriba en el eje Z para que no quede enterrado bajo el terreno.
  • 3Guarda el nivel con Ctrl + S antes de continuar.

Importar un Heightmap

En lugar de esculpir desde cero, puedes importar un archivo PNG o RAW de 16 bits desde herramientas como Gaea o World Machine usando la opción Import from File en el panel Manage.

3

Esculpir el Landscape (Sculpt Mode)

Activa el modo Sculpt para acceder a todas las herramientas de forma del terreno. Cada herramienta tiene parámetros de Brush Size, Brush Falloff y Tool Strength.

Vista del terreno con Sculpt Mode activo

Herramientas principales

⛰
Sculpt

Eleva o baja la altura del terreno. Herramienta principal para dar forma general.

Size: 8192 | Strength: 0.3
〰️
Smooth

Suaviza transiciones bruscas. Úsalo después de esculpir para naturalizar el terreno.

Size: 2048 | Strength: 0.3

Flatten

Aplana zonas a una misma elevación. Ideal para plataformas, caminos y poblados.

Genera una rampa suavizada entre dos puntos de distinta altura.

Width: 2000 | Side Falloff: 0.4

Herramientas de Erosión

Son costosas en terrenos grandes. Aplícalas en secciones pequeñas y usa Ctrl + Z si el resultado no es el esperado.

4

Materiales de Landscape y Paint Mode

Los Landscape Materials usan nodos exclusivos del Material Editor para gestionar múltiples capas de textura pintables sobre el terreno. El nodo central es LandscapeLayerBlend.

Estructura de carpetas recomendada: /Landscape/Materials, /Textures, /Levels

Crear el Material

  • 1En el Content Browser, crea un nuevo Material y ábrelo en el Material Editor.
  • 2Añade un nodo LandscapeLayerBlend y define las capas (ej. Grass, Rock, Dirt).
  • 3Para cada capa conecta sus texturas (Base Color, Normal, Roughness) a un nodo MakeMaterialAttributes.
  • 4Conecta el output del LayerBlend al nodo principal del material.
  • 5Asigna el material al Landscape Actor en el panel Details.
  • 6En Paint Mode, crea los archivos de capa haciendo clic en + junto a cada layer.

✦
Material Instances para tiling

Crea una Material Instance del material base y expón un parámetro de UV Scale. Así puedes ajustar el tiling de cada capa sin recompilar el shader.

Nodos clave del Material Editor

NodoFunción
LandscapeLayerBlendCombina capas de material usando máscaras pintadas. Nodo principal del flujo.
LandscapeLayerCoordsGenera UVs adaptados a la escala del terreno, evitando stretching.
LandscapeLayerSwitchActiva o desactiva una capa según si tiene pintura, optimizando el shader.
LandscapeLayerWeightExpone el peso de una capa como float para lógica personalizada.
5

Foliage Tool

La herramienta Foliage permite poblar el terreno con vegetación y objetos estáticos de forma masiva. Accede a ella desde Modes → Foliage.

Parámetros principales del Foliage Type

ParámetroFunciónValor sugerido
DensityInstancias por m² al pintar100–500 según tipo
Scale Min / MaxRango de escala aleatoriaMin 0.8 / Max 1.2
Random YawRotación aleatoria en YActivado siempre
Ground Slope AngleÁngulo máximo donde puede aparecer0–35° para árboles
Cull DistanceDistancia a la que se deja de renderizar5000–15000 UU
CollisionTipo de colisión del objetoNone para hierba
ℹ
PCG Framework

En UE 5.2+ puedes combinar Foliage con el Procedural Content Generation Framework para poblar el terreno automáticamente basándose en atributos como pendiente, altitud y capas de pintura.

⚠
Rendimiento

El exceso de foliage es una de las causas más comunes de bajo framerate. Usa Cull Distance y evita Collision en objetos pequeños como hierba o flores.
